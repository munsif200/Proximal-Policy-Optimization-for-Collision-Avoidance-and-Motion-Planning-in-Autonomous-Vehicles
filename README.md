# Proximal Policy Optimization for Collision Avoidance and Motion Planning in Autonomous Vehicles

## Publication Information

This repository contains the implementation and resources related to the research paper published in Fractals.

**DOI:** [10.1142/S0218348X25401711](https://doi.org/10.1142/S0218348X25401711)

**Paper Link:** [World Scientific - Fractals](https://www.worldscientific.com/doi/10.1142/S0218348X25401711)

## Abstract

This research explores the application of Proximal Policy Optimization (PPO), a state-of-the-art reinforcement learning algorithm, for collision avoidance and motion planning in autonomous vehicles.

## Overview

This repository implements PPO algorithms for:
- Collision avoidance in dynamic environments
- Motion planning for autonomous vehicles
- Safe navigation strategies using reinforcement learning
